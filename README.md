# 💰 Expense Tracker CLI

A simple command-line expense tracking application built with Python.

This project helps users record daily expenses, analyze spending habits, and generate reports directly from the terminal.

---

## 🚀 Features

* ➕ Add new expenses
* 📂 Categorize expenses (Food, Travel, Shopping, etc.)
* 📅 Track expenses by date
* 📊 Monthly spending summaries
* 🏷️ Category-wise spending analysis
* 📄 Export reports
* 📈 Average expense calculation
* 🔥 Top spending category detection
* 📉 Monthly spending trend analysis

---

## 🛠️ Technologies Used

* Python 3
* File Handling (CSV)
* Functions

* Modules

## ▶️ Running the Project

Clone the repository:

```bash
git clone https://github.com/your-username/expense-tracker-cli.git
```

Move into the project folder:

```bash
cd expense-tracker-cli
```

Run the application:

```bash
python main.py
```
